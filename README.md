# Maps API — mapa vivo

Bares, música ao vivo, happy hour e eventos — num mapa que se atualiza
sozinho todos os dias. Camada rica sobre o Google Maps, começando por São
Paulo e expandindo por demanda (Palmas e Goiânia já no ar).

**Abrir o mapa:** https://genio-da-lampada.github.io/maps-api-web/
**Estatísticas:** https://genio-da-lampada.github.io/maps-api-web/stats.html

Fonte dos dados: Maps API do [Gênio da Lâmpada](https://github.com/genio-da-lampada)
— Supabase + robô curador + Ticketmaster, Sympla, Ticket360, sites e
Instagram dos próprios lugares.
