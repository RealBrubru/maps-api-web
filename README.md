# Maps API — mapa vivo

Bares, música ao vivo e eventos de São Paulo, num mapa que se atualiza
sozinho todos os dias. Camada rica sobre o Google Maps.

**Abrir o mapa:** https://realbrubru.github.io/maps-api-web/

Fonte dos dados: Maps API (Supabase + robô curador + Ticketmaster, Sympla,
Ticket360 e sites dos próprios lugares).
